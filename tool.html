Android SDK

Android SDK可在多種作業系統，Windows、Linux、Mac等平台上開發Android相關應用的開發工具。包含了在Android平台上開發行動應用程式的各種工具集。

Android Studio

Android Studio為各種類型的Android提供最快速的開發工具，且支援Google雲端平台，讓開發者將自己的Android apps加入以雲端為基礎的後端，Google雲端平臺的後端會自動化擴展與管理，來因應數百萬個以上的用戶，替開發者省下購買、配置、維護與管理的需求。

Firebase

Firebase 是一個同時支援 Android、iOS 及網頁的 app 雲端開發平台，協助 app 開發者在雲端快速建置後端服務，提供即時資料庫，有效縮短 app開發時間，並幫助開發者更專注在前端的優化。

JSON

Firebase所提供的雲端資料庫其儲存格式為JSON，JSON是一種熱門的文字資料格式，用於在新式 Web 和行動應用程式中交換資料。 其也可用於將非結構化的資料儲存在記錄檔或是類似 Microsoft Azure Cosmos DB 的 NoSQL 資料庫中。

appear.in

appear.in是一個免費的高畫質線上多人視訊會議工具，免下載任何軟體，跨平台支援電腦版與Android、iOS App。只要進入網站中設定會議室的名稱，就會產生專屬的聊天室網址，可透過社群分享或複製網址連結，邀請成員加入聊天室。

  • 可以自訂視訊或語音通話房間的「專屬網址」
  • 不需要註冊、登入帳號
  • 不需要安裝其他軟體(需使用 Chrome、Firefox、Opera)
